Bienvenido a Eventos Salta

Nuestro proyecto consiste en una aplicación web que permite gestionar reservas de eventos a través de una interfaz de administración. Se pueden realizar operaciones CRUD en diferentes modulos como servicios, clientes, coordinadores y empleados, con la posibilidad de restaurar registros no activos. Además, se proporcionan API endpoints para acceder a la lista de servicios y al detalle de un servicio específico. Los usuarios podran revisar un práctico listado con todas las reservas realizadas por los clientes, manteniendo un registro claro y organizado gracias a la interfaz de la página que se presenta con un diseño sencillo y ordenado, facilitando la navegación y comprensión de las distintas funcionalidades disponibles.
